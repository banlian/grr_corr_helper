
## GRR CORR Helper

- grr自动生成：根据原始数据和grr模板格式，自动填充数据到grr模板文件
- corr自动生成：根据cmm测试数据和原始数据，自动填充数据到corr模板文件


- 内部文件夹
    - r'.\CMM' cmm 相关性数据文件夹
    - r'.\template' grr/corr 模板文件夹
    - r'.\Data' 数据文件夹
